# Spatial Chat Clone


Aims to create an open source replicate of https://spatial.chat/


## Try it out

### Dev
1. `make`
1. `make dev`

or

2. `make dev-d` to run it as daemon

Start client via "http://localhost:4500/ROOMID"

### Prod
1. `make`
1. `make prod`
